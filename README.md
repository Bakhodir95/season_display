# season_display

A Flutter package to display seasons and their attributes.

## Usage

```dart
import 'package:season_display/season_display.dart';

void main() {
  print(getSeasonName(Season.Spring)); // Bahor
  print(getSeasonColor(Season.Spring)); // Yoz
}
